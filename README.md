# OWMtgBot
Установка и настройка
```
git clone https://github.com/boif/OWMtgBot.git
cd OWMthBot
```

Создайте виртуальное окружение и активируйте его:
```
python3 -m venv venv
source venv/bin/activate
```

Установите зависимости:
```
pip install -r requirements.txt
```

Запуск бота:
```
python bot.py
```
